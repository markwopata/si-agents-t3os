# SI Agent Starter Figma Plugin

This is a local Figma development plugin for the SI Agent project.

It creates a starter canvas with four screens inspired by the current app:

- Dashboard
- Contacts
- Initiative detail
- Workbook import
- Knowledge view

## Import in Figma

1. Open the Figma desktop app.
2. Open any Design file.
3. Go to `Plugins > Development > Import new plugin from manifest...`
4. Select [manifest.json](/Users/mark.wopata/Documents/projects/SI-agent/figma/si-agent-starter/manifest.json)
5. Run `SI Agent Starter` from the Development section.
